## Production Notes

### Video Style Guidelines
- **Resolution**: Minimum 1080p (1920x1080)
- **Frame Rate**: 30 fps
- **Duration**: Keep each video under 10 minutes
- **Pacing**: Slow and clear, suitable for elderly audience and caregivers

### Visual Elements
- Use screen recordings with cursor highlights
- Add zoom effects for small UI elements
- Use arrows and annotations to guide attention
- Include captions/subtitles for accessibility

### Audio Guidelines
- Clear, professional voiceover
- Background music (subtle, non-distracting)
- Remove filler words ("um", "uh")
- Consistent volume levels

### Branding
- Start each video with SymBIoT logo (3 seconds)
- End with contact information and support resources
- Use brand colors consistently
- Include watermark in corner

### Accessibility
- Provide captions/subtitles
- Audio descriptions for visual elements
- Transcript available as text document
- Closed captions option

### Publishing Platforms
- YouTube (primary)
- Vimeo (backup)
- Embedded on website
- Mobile app help section
